#### Karma
* 模拟在真实的浏览器上测试。甚至不同的环境、os等
* 集成工作流，避免了在：编辑器--浏览器--页面间来回切换（效率低下）的流程。一个命令进行控制
* karma 不是一个测试框架，也没有断言库，只是运行了一个 http 服务，并生成一个测试入口 HTML 文件 Test Runner
